# 🤖 AI Resume Builder

An AI-powered Resume Builder that helps users create professional, ATS-friendly resumes with live preview, AI-generated summaries, PDF download, and secure Google Authentication.

## 🚀 Live Demo

https://ai-resume-builder-rpka.vercel.app/

## ✨ Features

- 🔐 Google Authentication using Firebase
- 📝 Resume Builder Form
- 👀 Live Resume Preview
- 🤖 AI Generated Professional Summary
- 📄 Download Resume as PDF
- 🖨️ Print Resume
- 💾 Save Resume to Firebase Firestore
- 📱 Responsive Design

## 🛠️ Tech Stack

- React.js
- Vite
- Firebase Authentication
- Firebase Firestore
- Google Gemini API
- HTML
- CSS
- JavaScript
